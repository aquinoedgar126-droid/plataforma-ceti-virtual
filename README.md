# Plataforma CETI Virtual

## Descripción

Proyecto desarrollado como parte de la carrera de Ingeniería en Tecnología de Software del Centro de Enseñanza Técnica Industrial (CETI).

El sistema consiste en una plataforma web integrada orientada a estudiantes de modalidad virtual, permitiendo centralizar información académica y administrativa mediante módulos de autenticación, horarios, seguimiento académico, pagos, avisos institucionales, preguntas frecuentes y contacto.
